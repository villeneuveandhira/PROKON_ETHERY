# PROKON_ETHERY
Aplikasi Supply Chain Berbasis Blockchain untuk Distribusi Buku

Situs supply-chain sederhana berbasis blockchain yang dirancang untuk transparansi dan keamanan.
Aplikasi ini memungkinkan pengguna (produsen, distributor, dan konsumen) untuk melacak setiap tahapan proses produksi hingga pengiriman secara real-time.
Dibangun dengan antarmuka web yang berinteraksi dengan kontrak pintar Ethereum melalui perpustakaan Ethers.js,
memungkinkan pengambilan data yang lancar dan manajemen transaksi yang aman di blockchain.
